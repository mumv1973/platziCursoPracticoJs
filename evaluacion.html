Una variable es un elemento(consideremos una caja) que sirve para almacenar datos (strings, números, bool, etc.), se dice que se almacena en un espacio en la memoria. ¿Cuál es la diferencia entre declarar e inicializar una variable? Declararla simplemente es nombrarla sin asignarle un valor aún. Ej. var num ; Inicializar es cuando ya se le asigna un valor usando el = con algún dato sea strings, números, bool, etc. Ej. var num = 5; ¿Cuál es la diferencia entre sumar números y concatenar strings? No entendí bien, pero sumar números es una operación algebraica Ej. 3 + 5 = 8 Contatenar nos sirve para “unir” strings uno detrás de otro.
